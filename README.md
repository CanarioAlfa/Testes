# Testes

Dois exemplos de apps mobile de academia focados em vídeo/GIF para execução correta, registro de peso e integração com smartwatches (Mi Band e outros):

## Gym Lite (apps/gym-lite)
- Lista de exercícios com vídeo ou GIF, descrição de grupamento muscular e dicas rápidas.
- Registro de peso com histórico local e destaque do último lançamento.
- Toggle de conexão com Mi Band para simular sincronização de biometria.

## Gym Pro (apps/gym-pro)
- Painel de métricas (carga semanal, sono, batimentos) conectado a wearables.
- Exercícios com dicas baseadas em sensores (HRV, velocidade, amplitude).
- Registro de peso com tendência, planejamento diário com readiness e recomendações.
- Controles separados para conexão e sincronização automática de smartwatches.

Ambos os apps usam React Native + Expo. Rode `npm install` dentro de cada pasta e depois `npm run start` para abrir o Metro bundler.
